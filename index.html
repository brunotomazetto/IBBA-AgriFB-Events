Setup Guide

1

Create a free Supabase account

Go to supabase.com → Sign up → New Project. Choose a name and region close to you (e.g. South America).

2

Create the database tables

In your Supabase project go to SQL Editor and run the query below:

create table events (
  id text primary key,
  name text not null,
  speaker text, sector text,
  registered_date date,
  requested_date date,
  fup_date date,
  scheduled_date date,
  obs text,
  canceled boolean default false,
  rating_pre_call int default 0,
  rating_call int default 0,
  updated_at timestamptz default now()
);

create table speakers (
  id text primary key,
  name text not null,
  company text, sector text, email text
);

alter table events enable row level security;
alter table speakers enable row level security;
create policy "allow all" on events for all using (true);
create policy "allow all" on speakers for all using (true);
3

Copy your credentials

Go to Project Settings → API. Copy the Project URL and the anon public key. Paste them in the form on the left.

4

Share the dashboard link

Send https://brunotomazetto.github.io/IBBA-AgriFB-Events/ to your colleagues. Everyone connecting with their own login will see and edit the same data in real time.
